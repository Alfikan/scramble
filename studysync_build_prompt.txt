PROJECT NAME: Scramble

PROJECT TYPE: Full-stack collaborative study platform web application
TARGET USERS: Students, friends, and classmates who want to study together
CORE CONCEPT: Room-based study collaboration similar to Google Classroom with dual modes (Private Space & Group Study)

================================================================================
                              TECH STACK
================================================================================

FRONTEND:
- React 18+ with Hooks
- React Router v6 (client-side routing)
- Tailwind CSS (utility-first styling)
- Framer Motion (animations)
- React Hook Form + Yup (forms & validation)
- React Query (data fetching & caching)
- Socket.io-client (real-time features)
- Recharts (data visualization)
- Draft.js or Slate.js (rich text editor)
- KaTeX (LaTeX math rendering)

BACKEND:
- Firebase Authentication (email/password, Google OAuth, Microsoft OAuth)
- Cloud Firestore (NoSQL database)
- Cloud Storage (media & file storage)
- Cloud Functions (serverless backend logic)
- Firebase Realtime Database (presence & online status)
- Firebase Cloud Messaging (push notifications)

THIRD-PARTY APIs:
- Agora.io or Daily.co (video calling, screen sharing, recording)
- OpenAI API or Google Gemini (AI quiz generation, doubt assistance)
- Google Calendar API (schedule sync)

================================================================================
                            DESIGN SYSTEM
================================================================================

TYPOGRAPHY:
Font Family: Kodchasan (Google Fonts)
- Load weights: 300, 400, 500, 600, 700

Usage:
- Page Titles / Room Names: Kodchasan Bold (700), 28-32px
- Section Headings: Kodchasan SemiBold (600), 20-24px
- Body Text / Content: Kodchasan Medium (500), 15-16px
- Secondary Text / Metadata: Kodchasan Regular (400), 13-14px
- Small Labels / Captions: Kodchasan Light (300), 11-12px

COLOR PALETTE:
Primary Colors:
- Deep Black: #151313 (sidebar, primary text, headers)
- Vibrant Orange: #ff5734 (CTAs, active states, notifications)
- Soft Purple: #be94f5 (group study mode, quizzes)
- Warm Yellow: #fccc42 (private mode, achievements, timers)
- Light Cream: #f7f7f5 (card backgrounds, input fields)
- Cool Blue-Gray: #d4e4e8 (main background)

Semantic Colors:
- Success Green: #4caf50 (completed tasks, correct answers)
- Warning Red: #f44336 (deadlines, errors, incorrect answers)
- Info Blue: #2196f3 (notifications, informational messages, links)
- Muted Gray: #9e9e9e (disabled states, placeholders)

Feature-Specific Color Mapping:
- Private Space features: Yellow (#fccc42) accents
- Group Study features: Purple (#be94f5) accents
- Study Timer: Yellow with orange progress
- Quizzes: Purple cards with orange buttons
- Doubts (Open): Orange badges
- Doubts (Resolved): Green badges
- Active Meetings: Blue (#2196f3)

DESIGN PRINCIPLES:
- Card-based UI with rounded corners (24px radius)
- Generous white space and padding (24-32px inside cards)
- Soft shadows on cards: box-shadow: 0 2px 8px rgba(0,0,0,0.08)
- Hover effects: transform: translateY(-4px) with deeper shadow
- Button border-radius: 20px
- Input border-radius: 12px
- Pills/Tags border-radius: 20px
- Smooth transitions: transition: all 0.3s ease

================================================================================
                        DATABASE STRUCTURE (FIRESTORE)
================================================================================

COLLECTION: users
  DOCUMENT: {userId}
    FIELDS:
      - displayName: string
      - email: string
      - avatar: string (Storage URL)
      - bio: string (optional)
      - subjects: array of strings
      - studyPreferences: object
          * focusMusic: boolean
          * pomodoroLength: number (minutes)
          * breakLength: number (minutes)
          * studyGoalHoursPerWeek: number
      - availability: object
          * monday: array [startHour, endHour]
          * tuesday: array [startHour, endHour]
          * ... (for each day of week)
      - stats: object
          * totalStudyHours: number
          * totalQuizzesTaken: number
          * averageQuizScore: number
          * doubtsRaised: number
          * doubtsResolved: number
          * meetingsAttended: number
          * currentStreak: number
          * longestStreak: number
          * xp: number
          * level: number
      - badges: array of strings (badge IDs)
      - joinedRooms: array of room IDs
      - createdAt: timestamp
      - lastActive: timestamp

COLLECTION: rooms
  DOCUMENT: {roomId}
    FIELDS:
      - name: string
      - description: string
      - code: string (6-digit unique code)
      - subject: string
      - adminId: string (user ID of creator)
      - members: array of user IDs
      - memberLimit: number (5-50)
      - privacy: string ("public", "private", "restricted")
      - settings: object
          * allowMemberInvites: boolean
          * autoArchiveInactiveDays: number
          * defaultTimerMinutes: number
      - stats: object
          * totalStudyHours: number
          * totalQuizzes: number
          * totalResources: number
          * totalMeetings: number
      - createdAt: timestamp
      - lastActivity: timestamp

  SUBCOLLECTION: messages
    DOCUMENT: {messageId}
      FIELDS:
        - senderId: string (user ID)
        - senderName: string
        - senderAvatar: string
        - content: string
        - type: string ("text", "file", "image", "system", "poll")
        - attachments: array of objects
            * name: string
            * url: string (Storage URL)
            * type: string (file type)
            * size: number (bytes)
        - replyTo: string (message ID, optional)
        - reactions: object (emoji: array of user IDs)
        - mentions: array of user IDs
        - isPinned: boolean
        - timestamp: timestamp
        - editedAt: timestamp (optional)

  SUBCOLLECTION: resources
    DOCUMENT: {resourceId}
      FIELDS:
        - name: string
        - type: string ("pdf", "image", "video", "link", "doc")
        - url: string (Storage URL or external link)
        - uploadedBy: string (user ID)
        - uploadedByName: string
        - size: number (bytes, if file)
        - folderId: string (category folder, optional)
        - tags: array of strings
        - description: string (optional)
        - comments: array of objects
            * userId: string
            * userName: string
            * comment: string
            * timestamp: timestamp
        - reactions: object (type: count)
        - uploadedAt: timestamp

  SUBCOLLECTION: tasks
    DOCUMENT: {taskId}
      FIELDS:
        - title: string
        - description: string (optional)
        - createdBy: string (user ID)
        - assignedTo: array of user IDs (for group tasks)
        - priority: string ("low", "medium", "high")
        - dueDate: timestamp
        - subject: string (optional)
        - completed: boolean
        - completedBy: string (user ID, optional)
        - completedAt: timestamp (optional)
        - createdAt: timestamp

COLLECTION: doubts
  DOCUMENT: {doubtId}
    FIELDS:
      - title: string
      - description: string (rich text HTML)
      - raisedBy: string (user ID)
      - raisedByName: string
      - raisedByAvatar: string
      - roomId: string
      - roomName: string
      - subject: string
      - priority: string ("low", "medium", "high")
      - status: string ("open", "in_progress", "resolved")
      - tags: array of strings
      - attachments: array of objects
          * name: string
          * url: string
          * type: string
      - visibility: string ("room", "specific") - if specific, visibleTo array
      - visibleTo: array of user IDs (optional)
      - resolvedBy: string (user ID, optional)
      - bestAnswerId: string (comment ID, optional)
      - upvotes: number
      - createdAt: timestamp
      - updatedAt: timestamp
      - resolvedAt: timestamp (optional)

  SUBCOLLECTION: comments
    DOCUMENT: {commentId}
      FIELDS:
        - userId: string
        - userName: string
        - userAvatar: string
        - content: string (rich text HTML, supports code, LaTeX)
        - replyTo: string (comment ID for threaded replies, optional)
        - upvotes: array of user IDs
        - timestamp: timestamp
        - editedAt: timestamp (optional)

COLLECTION: quizzes
  DOCUMENT: {quizId}
    FIELDS:
      - title: string
      - topic: string
      - description: string (optional)
      - createdBy: string (user ID)
      - roomId: string (if group quiz, null for personal)
      - type: string ("personal", "group", "sprint")
      - difficulty: string ("easy", "medium", "hard")
      - timeLimit: number (seconds per question, 0 for no limit)
      - questions: array of objects
          * questionText: string
          * questionType: string ("mcq", "true_false", "fill_blank", "short_answer")
          * options: array of strings (for mcq)
          * correctAnswer: string or number (option index for mcq)
          * explanation: string
          * points: number (default 1)
      - totalQuestions: number
      - totalPoints: number
      - participants: array of objects (for group quizzes)
          * userId: string
          * userName: string
          * score: number
          * completedAt: timestamp
      - scheduledAt: timestamp (optional)
      - isActive: boolean
      - createdAt: timestamp

COLLECTION: quizResults
  DOCUMENT: {resultId}
    FIELDS:
      - quizId: string
      - userId: string
      - userName: string
      - userAvatar: string
      - roomId: string (optional)
      - score: number
      - totalPoints: number
      - percentage: number
      - timeTaken: number (seconds)
      - answers: array of objects
          * questionId: number (index)
          * userAnswer: string
          * isCorrect: boolean
          * pointsEarned: number
      - completedAt: timestamp

COLLECTION: studySessions
  DOCUMENT: {sessionId}
    FIELDS:
      - userId: string
      - subject: string
      - duration: number (minutes)
      - sessionType: string ("pomodoro", "deep_focus", "custom")
      - startTime: timestamp
      - endTime: timestamp
      - completed: boolean (false if interrupted)
      - focusModeEnabled: boolean
      - ambientSound: string (optional)

COLLECTION: meetings
  DOCUMENT: {meetingId}
    FIELDS:
      - title: string
      - roomId: string
      - createdBy: string (user ID)
      - scheduledTime: timestamp
      - duration: number (minutes)
      - agenda: string (optional)
      - participants: array of user IDs (invited)
      - attendees: array of objects (who actually joined)
          * userId: string
          * joinedAt: timestamp
          * leftAt: timestamp
      - resources: array of resource IDs (attached files)
      - recordingUrl: string (Storage URL, optional)
      - isRecurring: boolean
      - recurrencePattern: string ("daily", "weekly", null)

service cloud.firestore {
  match /databases/{database}/documents {
    
    // Helper functions
    function isSignedIn() {
      return request.auth != null;
    }
    
    function isOwner(userId) {
      return request.auth.uid == userId;
    }
    
    function isRoomMember(roomId) {
      return request.auth.uid in get(/databases/$(database)/documents/rooms/$(roomId)).data.members;
    }
    
    function isRoomAdmin(roomId) {
      return request.auth.uid == get(/databases/$(database)/documents/rooms/$(roomId)).data.adminId;
    }
    
    // Users collection
    match /users/{userId} {
      allow read: if isSignedIn();
      allow create: if isSignedIn();
      allow update: if isOwner(userId);
      allow delete: if isOwner(userId);
    }
    
    // Rooms collection
    match /rooms/{roomId} {
      allow read: if isSignedIn() && (
        resource.data.privacy == "public" || 
        isRoomMember(roomId)
      );
      allow create: if isSignedIn();
      allow update: if isRoomAdmin(roomId);
      allow delete: if isRoomAdmin(roomId);
      
      // Messages subcollection
      match /messages/{messageId} {
        allow read: if isRoomMember(roomId);
        allow create: if isRoomMember(roomId);
        allow update: if isRoomMember(roomId) && (
          request.auth.uid == resource.data.senderId || 
          isRoomAdmin(roomId)
        );
        allow delete: if isRoomAdmin(roomId);
      }
      
      // Resources subcollection
      match /resources/{resourceId} {
        allow read: if isRoomMember(roomId);
        allow create: if isRoomMember(roomId);
        allow update: if isRoomMember(roomId);
        allow delete: if isRoomMember(roomId) && (
          request.auth.uid == resource.data.uploadedBy || 
          isRoomAdmin(roomId)
        );
      }
      
      // Tasks subcollection
      match /tasks/{taskId} {
        allow read: if isRoomMember(roomId);

        allow delete: if isRoomAdmin(roomId);

      match /comments/{commentId} {
        allow read: if isSignedIn();
        allow create: if isSignedIn();

        allow delete: if isOwner(resource.data.userId);
      }
    }
    
    // Quizzes collection
    match /quizzes/{quizId} {
      allow read: if isSignedIn() && (
        resource.data.roomId == null || 
        isRoomMember(resource.data.roomId)
      );
      allow create: if isSignedIn();
      allow update: if isOwner(resource.data.createdBy);
      allow delete: if isOwner(resource.data.createdBy);
    }
    
    // Quiz Results collection
    match /quizResults/{resultId} {
      allow read: if isSignedIn() && (
        isOwner(resource.data.userId) || 
        isRoomMember(resource.data.roomId)
      );
      allow create: if isSignedIn();
      allow update: if false; // Results are immutable
      allow delete: if isOwner(resource.data.userId);
    }
    
    // Study Sessions collection
    match /studySessions/{sessionId} {
      allow read: if isOwner(resource.data.userId);
      allow create: if isSignedIn();
      allow update: if isOwner(resource.data.userId);
      allow delete: if isOwner(resource.data.userId);
    }
    
    // Meetings collection
    match /meetings/{meetingId} {
      allow read: if isSignedIn() && (
        request.auth.uid in resource.data.participants ||
        isRoomMember(resource.data.roomId)
      );
      allow create: if isSignedIn();
      allow update: if isOwner(resource.data.createdBy);

      allow update: if isOwner(resource.data.userId);
      allow delete: if isOwner(resource.data.userId);
    }

    match /exams/{examId} {
      allow read: if isSignedIn() && (
        isOwner(resource.data.userId) ||

      );
      allow create: if isSignedIn();
      allow update: if isOwner(resource.data.userId);
      allow delete: if isOwner(resource.data.userId);

exports.generateRoomCode = functions.https.onCall(async (data, context) => {
  if (!context.auth) {
    throw new functions.https.HttpsError('unauthenticated', 'Must be signed in');
  }
  
  // Generate random 6-digit alphanumeric code
  const chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789';
  let code;
  let isUnique = false;
  
  while (!isUnique) {
    code = '';
    for (let i = 0; i < 6; i++) {
      code += chars.charAt(Math.floor(Math.random() * chars.length));
    }
    
    // Check if code already exists
    const snapshot = await db.collection('rooms').where('code', '==', code).get();
    if (snapshot.empty) {
      isUnique = true;
    }
  }
  
  return { code };
});

exports.sendMeetingReminders = functions.pubsub
  .schedule('every 5 minutes')
  .onRun(async (context) => {
    const now = admin.firestore.Timestamp.now();
    const fifteenMinutesLater = new admin.firestore.Timestamp(
      now.seconds + (15 * 60),
      now.nanoseconds
    );
    
    // Find meetings starting in the next 15 minutes that haven't sent 15-min reminder
    const meetingsSnapshot = await db.collection('meetings')
      .where('scheduledTime', '>', now)
      .where('scheduledTime', '<=', fifteenMinutesLater)
      .where('status', '==', 'scheduled')
      .get();
    
    const batch = db.batch();
    
    for (const doc of meetingsSnapshot.docs) {
      const meeting = doc.data();
      
      // Check if 15-minute reminder was already sent
      const reminderSent = meeting.reminders && 
                          meeting.reminders.some(r => r.toMillis() === fifteenMinutesLater.toMillis());
      
      if (!reminderSent) {
        // Send notifications to all participants
        for (const participantId of meeting.participants) {
          const notificationRef = db.collection('notifications').doc();
          batch.set(notificationRef, {
            userId: participantId,
            type: 'meeting',
            priority: 'urgent',
            title: 'Meeting starting soon',
            message: `"${meeting.title}" starts in 15 minutes`,
            actionUrl: `/meetings/${doc.id}`,
            actionData: { meetingId: doc.id },
            isRead: false,
            createdAt: admin.firestore.FieldValue.serverTimestamp()
          });
        }
        
        // Mark reminder as sent
        batch.update(doc.ref, {
          reminders: admin.firestore.FieldValue.arrayUnion(fifteenMinutesLater)
        });
      }
    }
    
    await batch.commit();
    return null;
  });

exports.updateLeaderboard = functions.pubsub
  .schedule('every 1 hours')
  .onRun(async (context) => {
    const roomsSnapshot = await db.collection('rooms').get();
    
    for (const roomDoc of roomsSnapshot.docs) {
      const roomId = roomDoc.id;
      const members = roomDoc.data().members;
      
      const leaderboardData = [];
      
      for (const memberId of members) {
        // Get user's study time in this room
        const sessionsSnapshot = await db.collection('studySessions')
          .where('userId', '==', memberId)
          .get();
        
        let totalStudyMinutes = 0;
        sessionsSnapshot.forEach(doc => {
          totalStudyMinutes += doc.data().duration;
        });
        
        // Get user's quiz scores in this room
        const quizResultsSnapshot = await db.collection('quizResults')
          .where('userId', '==', memberId)
          .where('roomId', '==', roomId)
          .get();
        
        let totalQuizScore = 0;
        let quizCount = 0;
        quizResultsSnapshot.forEach(doc => {
          totalQuizScore += doc.data().percentage;
          quizCount++;
        });
        
        const avgQuizScore = quizCount > 0 ? totalQuizScore / quizCount : 0;
        
        // Calculate composite score (weighted: 60% study time, 40% quiz performance)
        const studyScore = totalStudyMinutes / 60; // hours
        const compositeScore = (studyScore * 0.6) + (avgQuizScore * 0.4);
        
        const userDoc = await db.collection('users').doc(memberId).get();
        const user = userDoc.data();
        
        leaderboardData.push({
          userId: memberId,
          userName: user.displayName,
          userAvatar: user.avatar,
          studyHours: studyScore,
          avgQuizScore: avgQuizScore,
          compositeScore: compositeScore
        });
      }
      
      // Sort by composite score
      leaderboardData.sort((a, b) => b.compositeScore - a.compositeScore);
      
      // Store leaderboard (create a new subcollection or field in room document)
      await roomDoc.ref.update({
        leaderboard: leaderboardData
      });
    }
    
    return null;
  });

   exports.generateThumbnail = functions.storage.object().onFinalize(async (object) => {
     const filePath = object.name;
     const contentType = object.contentType;
     
     if (!contentType.startsWith('image/')) {
       return null;
     }
     
     if (filePath.includes('thumb_')) {
       return null; // Already a thumbnail
     }
     
     const bucket = admin.storage().bucket(object.bucket);
     const fileName = filePath.split('/').pop();
     const thumbFileName = `thumb_${fileName}`;
     const thumbFilePath = filePath.replace(fileName, thumbFileName);
     
     const tempFilePath = `/tmp/${fileName}`;
     const tempThumbPath = `/tmp/${thumbFileName}`;
     
     await bucket.file(filePath).download({ destination: tempFilePath });
     
     await sharp(tempFilePath)
       .resize(200, 200, { fit: 'cover' })
       .toFile(tempThumbPath);
     
     await bucket.upload(tempThumbPath, {
       destination: thumbFilePath,
       metadata: {
         contentType: contentType,
       },
     });
     
     return null;
   });

   // hooks/useDebounce.js
   import { useState, useEffect } from 'react';
   
   export const useDebounce = (value, delay = 500) => {
     const [debouncedValue, setDebouncedValue] = useState(value);
     
     useEffect(() => {
       const handler = setTimeout(() => {
         setDebouncedValue(value);
       }, delay);
       
       return () => {
         clearTimeout(handler);
       };
     }, [value, delay]);
     
     return debouncedValue;
   };
   
   // Usage in search input
   const SearchBar = () => {

         performSearch(debouncedSearchTerm);

     }, [debouncedSearchTerm]);
     
     return (

         type="text"
         value={searchTerm}
         onChange={(e) => setSearchTerm(e.target.value)}

       />
     );

// Example: Accessible Modal
const AccessibleModal = ({ isOpen, onClose, title, children }) => {
  const modalRef = useRef();
  const closeButtonRef = useRef();
  
  useEffect(() => {
    if (isOpen) {
      closeButtonRef.current?.focus();
    }
  }, [isOpen]);
  
  const handleKeyDown = (e) => {
    if (e.key === 'Escape') {
      onClose();
    }
  };
  
  if (!isOpen) return null;
  
  return (
    <div
      className="modal-overlay"
      role="dialog"
      aria-modal="true"
      aria-labelledby="modal-title"
      onKeyDown={handleKeyDown}
      ref={modalRef}
    >
      <div className="modal-content">
        <div className="modal-header">
          <h2 id="modal-title">{title}</h2>
          <button
            ref={closeButtonRef}
            onClick={onClose}
            aria-label="Close modal"
            className="close-btn"
          >
            ×
          </button>
        </div>
        <div className="modal-body">
          {children}
        </div>
      </div>
    </div>
  );
};

   // public/service-worker.js
   const CACHE_NAME = 'studysync-v1';
   const urlsToCache = [
     '/',
     '/index.html',
     '/static/css/main.css',
     '/static/js/main.js',
     '/manifest.json',
     '/logo192.png',
     '/logo512.png',
   ];
   
   // Install service worker
   self.addEventListener('install', (event) => {
     event.waitUntil(
       caches.open(CACHE_NAME)
         .then((cache) => cache.addAll(urlsToCache))
     );
   });
   
   // Fetch from cache first, then network
   self.addEventListener('fetch', (event) => {
     event.respondWith(
       caches.match(event.request)
         .then((response) => {
           if (response) {
             return response;
           }
           return fetch(event.request);
         })
     );
   });
   
   // Delete old caches
   self.addEventListener('activate', (event) => {
     event.waitUntil(
       caches.keys().then((cacheNames) => {
         return Promise.all(
           cacheNames.map((cacheName) => {
             if (cacheName !== CACHE_NAME) {
               return caches.delete(cacheName);
             }
           })
         );
       })
     );
   });
